# PLAN — 급여자동계산기

> COAT Phase: PLAN
> 작성일: 2026-04-10
> 작성자: 기획자

---

## 핵심 목표

직원의 기본급·수당 입력만으로 **실수령액과 4대보험·세금 공제 내역을 즉시 자동 계산**해주는 월급 계산기를 제공한다.

---

## 요구사항

### 필수 (Must)
- [ ] 기본급 + 수당(야근·식대·교통비 등) 입력
- [ ] 4대보험 자동 계산
  - 국민연금 (근로자 부담분 4.5%)
  - 건강보험 (근로자 부담분 3.545%)
  - 장기요양보험 (건강보험료 × 12.95%)
  - 고용보험 (근로자 부담분 0.9%)
- [ ] 근로소득세 자동 계산 (간이세액표 기반)
- [ ] 지방소득세 자동 계산 (근로소득세 × 10%)
- [ ] 실수령액 자동 계산 (총 급여 - 총 공제액)
- [ ] 계산 결과 화면 표시 (공제 항목별 금액 명세)

### 필수 추가 (Must — 2차 확정)
- [ ] **스케줄제 모드** — 시급 × 근무시간 기반 급여 자동 산출
  - **근무 달력 UI** — 월별 캘린더 화면에서 날짜별 출퇴근 시간 직접 입력
    - 근무한 날: 출근 시간 ~ 퇴근 시간 표시 (예: `09:00 ~ 18:00`)
    - 연차 사용일: 별도 색상 표시
    - 조퇴일: 단축 시간 표시 (예: `09:00 ~ 14:00 ✂`)
    - 공휴일·주말 자동 구분 표시
  - 일별 근무 스케줄 입력 (출근·퇴근·휴게 시간)
  - 연장·야간·휴일근로 자동 구분 및 할증 적용 (1.5배/2.0배)
  - **연차** — 사용 시 유급 처리 (1일 = 소정근로시간 × 시급), 잔여 연차 자동 차감
  - **조퇴** — 실제 퇴근 시간 기준 단축 근무시간 자동 계산, 해당 시간분 급여 차감
- [ ] **주휴수당 계산** — 주 15시간 이상 개근 시 자동 산정
  - 주휴수당 = (주 소정근로시간 / 40) × 8 × 시급
- [ ] **연말정산 예상 계산** — 연간 누적 급여 기반 환급/추가납부 예상액
  - 근로소득공제, 인적공제(본인·부양가족), 기본 특별공제 적용
  - 월별 기납부 세액 대비 차액 계산
- [ ] **실업급여 실시간 계산** — 퇴직 시 수령 가능한 구직급여 즉시 산출
  - 기준 일급 (이직 전 3개월 평균임금 × 60%)
  - 소정급여일수 (나이·고용보험 가입기간 테이블 기반)
  - 상한액(66,000원/일) · 하한액(최저임금 80%) 자동 적용
- [ ] **퇴직금 계산** — 계속 근로 1년 이상 시 법정 퇴직금 산출
  - 평균임금 = 퇴직 전 3개월 총 임금 / 해당 기간 총 일수
  - 퇴직금 = 평균임금 × 30일 × (재직일수 / 365)

### 설정 화면 (Settings)
- [ ] **시간 표시 형식** — 24시간제 / 12시간제(AM·PM) 선택
  - 설정 화면 토글 ON(24시간) / OFF(12시간)
  - 달력 근무 시간 표시 및 출퇴근 입력 시간 피커 모두 동일 형식 적용
  - 기본값: 24시간제

### 선택 (Should)
- [ ] 부양가족 수 입력에 따른 세금 조정
- [ ] 비과세 항목 분리 처리 (식대 20만 원, 차량유지비 20만 원 등)
- [ ] 계산 결과 PDF 또는 텍스트 내보내기

### 향후 고려 (Could)
- [ ] 연간 연봉 → 월급 환산
- [ ] 회사 부담분(사용자 부담 4대보험) 함께 표시
- [ ] **회사 앱 연동** — 외부 근태·HR 시스템과 데이터 연동
  - REST API 또는 CSV/Excel 가져오기 방식 (구현 가능성 탐색 필요)
  - 연동 대상 앱 확정 후 설계 단계에서 구체화

## 성공 기준

1. **정확도** — 기본급 3,000,000원 입력 시 4대보험·소득세 계산값이 2025년 공식값과 ±100원 이내 일치
2. **속도** — 숫자 입력 후 3초 내 결과 확인 (실시간 계산)
3. **UX** — 공제 항목명·금액이 명세서 형태로 한눈에 보임
4. **스케줄 모드** — 일별 근무표 입력 후 주휴수당 포함 실수령액 자동 계산
5. **연말정산** — 월별 기납부 세액 누적 후 환급/추납 예상액 ±5만 원 이내 정확도
6. **실업급여** — 입력 즉시 구직급여 총액·일수·일급 3가지 항목 표시
7. **퇴직금** — 입사일·퇴사일·최근 3개월 급여만 입력하면 퇴직금 즉시 산출

## 기술 스택 (초안)

- **최종 목표 플랫폼**: Android + iOS 네이티브 앱
- **프레임워크 후보** (DESIGN 단계에서 확정):
  | 후보 | 장점 | 단점 |
  |------|------|------|
  | **React Native** | JS/TS 기반, 생태계 풍부, 웹 개발자 친화 | 복잡한 UI 성능 주의 |
  | **Flutter** | 성능 우수, UI 일관성, 단일 코드베이스 | Dart 언어 학습 필요 |
- **권장**: React Native (TypeScript) — 회사 앱 연동 시 웹 API 연계 용이
- **외부 API**: 없음 (순수 계산 로직, 오프라인 동작 가능)
- **스토어 배포**: Google Play Store + Apple App Store
