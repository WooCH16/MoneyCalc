## 계산 로직 모듈 (src/utils/)

```
src/
└── utils/
    ├── insuranceCalc.ts      — 4대보험 (국민연금·건강보험·장기요양·고용보험)
    ├── taxCalc.ts            — 간이세액표 기반 근로소득세 + 지방소득세
    ├── scheduleCalc.ts       — 스케줄제 급여·연장·야간·휴일 할증
    ├── holidayPayCalc.ts     — 주휴수당 (주 15h 이상 개근 판정 포함)
    ├── leaveCalc.ts          — 연차 유급처리·조퇴 차감
    ├── yearEndCalc.ts        — 연말정산 예상 (누적 기납부세액 대비)
    ├── unemploymentCalc.ts   — 실업급여 (상한·하한·소정급여일수 테이블)
    ├── severanceCalc.ts      — 퇴직금 (평균임금 × 30 × 재직비율)
    └── formatters.ts         — 금액 콤마 포맷·시간 24h/12h 변환
```

## 데이터 흐름

```
[사용자 입력]
     │
     ▼
[Zustand Store]
  ├── settingsStore   — 시간제(24h/12h), 기준 시급, 소정근로시간
  ├── scheduleStore   — 월별 날짜별 근무 데이터
  └── calcStore       — 계산 결과 캐시
     │
     ▼
[계산 유틸 호출] ──────────────────────────────┐
  insuranceCalc / taxCalc / scheduleCalc 등    │
     │                                         │
     ▼                                         ▼
[ResultScreen / SalaryResultScreen]     [ScheduleScreen 달력 배지]

[AsyncStorage] ← scheduleStore 자동 persist (월별 저장)
```
